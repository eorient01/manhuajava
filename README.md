本书以漫画的形式介绍了 Java 语言的相关知识。 全书共 20 章， 书中首先介绍了 Java 语言的历史、
特点、 开发环境、 运算符、 数据类型、 数组、 字符串、 判断语句、 循环语句等基础知识， 接着讨论了函
数式编程、 类与对象、 异常、 文件操作、 集合、 图形用户界面、 网络通信、 多线程等进阶内容， 最后通过一个案例向大家介绍如何爬取图片。
